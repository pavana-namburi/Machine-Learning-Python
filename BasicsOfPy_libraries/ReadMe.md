# Experiment 01 ‚Äì NumPy and Pandas Fundamentals

## Objective

The objective of this experiment is to understand the foundational concepts of NumPy and Pandas, which are essential libraries for data analysis and machine learning in Python.

This experiment focuses on array operations, data manipulation, handling missing values, and performing basic data analysis tasks.

---

## üìÅ Files in This Experiment

- `01_NumPy_Fundamentals.ipynb`  
  Covers NumPy array creation, indexing, slicing, reshaping, iteration, and array operations.

- `02_Pandas_Fundamentals.ipynb`  
  Covers Series and DataFrame creation, CSV loading, data inspection, missing value handling, feature engineering, and filtering operations.

- `data/students.csv`  
  Sample dataset used for practicing data manipulation and analysis.

---

## Concepts Covered

### NumPy
- 0D, 1D, 2D, and 3D array creation
- Shape and dimension analysis
- Indexing and slicing
- Data types and type conversion
- Copy vs View
- Reshaping and flattening
- Iteration techniques
- Concatenation and stacking
- Pattern creation using slicing

### Pandas
- Creating Series from lists and dictionaries
- Creating DataFrames
- Loading CSV files
- Inspecting dataset structure
- Handling missing values using mean imputation
- Feature engineering (Total, Average, Result)
- Conditional filtering operations

## Dataset Description

The `students.csv` dataset contains:

- `Student_ID`

- `Branch`

The dataset includes missing values that are handled using mean imputation before performing analysis.
